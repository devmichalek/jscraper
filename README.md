## Build dependencies
1. [Rust](https://doc.rust-lang.org/book/ch01-01-installation.html)
2. [Docker engine](https://docs.docker.com/engine/install/)
3. [Make](https://www.gnu.org/software/make/)

## Ideas
1. run `rustfmt <sources>` to format files
2. run `cargo check` and then `cargo build`
3. how to benchamark the binary? use `RUST_BACKTRACE=1 cargo run`
4. build release version `cargo build --release`
